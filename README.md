#  Sorting Algorithms Visualizer

An interactive and visually appealing **Sorting Algorithms Visualizer** built using **React**, **Tailwind CSS**, and **Framer Motion**.  
It allows users to explore and understand how popular sorting algorithms work step-by-step through real-time animations.
